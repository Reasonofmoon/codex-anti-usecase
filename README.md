# AI Use-Cases & Anti-Use Cases Dashboard

이 저장소는 **OpenAI Codex 유스케이스(60개)**, Claude Code용 **Antigravity 플러그인 유스케이스(23개)**, 그리고 AI 도입 실패 및 부적합 시나리오를 정의한 **AI 안티 유스케이스(15개)**를 한국어로 통합 분석하여 효과적으로 열람할 수 있는 **프리미엄 반응형 대시보드 웹 애플리케이션**입니다.

---

## 🎨 대시보드 스크린샷 및 주요 특징

- **3중 멀티 컬렉션 토글**: 상단 글래스모피즘 스위치를 통해 3가지 데이터셋을 자유롭게 넘나들 수 있습니다.
- **실시간 통계 카운트**: 데이터셋 교체 시 숫자가 부드럽게 롤링 업(`animateValue`)하는 마이크로 인터랙션을 가집니다.
- **동적 필터 생성**: 각 데이터셋에 맞는 카테고리(정밀도, 효율성, 시스템 등)가 동적으로 탭 버튼으로 렌더링됩니다.
- **터미널 커맨드 코드 스타일**: 플러그인 유스케이스의 슬래시 명령어들을 Monospace 테마와 네온 블루 아웃라인으로 코드 뱃지 렌더링했습니다.
- **💡 창의적 응용 제안 (Insights)**: 모달 하단에 저농도 발광 효과가 적용된 박스로 각 항목에 대한 비즈니스 대안 설계를 추천합니다.

## 🚀 로컬 대시보드 구동 방법

로컬에서 비동기 fetch를 지원하기 위해 간이 웹 서버를 통해 구동해야 정상 동작합니다.

### 1. Python 내장 서버 구동
저장소 폴더 위치에서 터미널을 열고 다음 명령을 실행합니다:
```bash
python -m http.server 8000
```
이후 웹 브라우저에서 **[http://localhost:8000](http://localhost:8000)**에 접속합니다.

### 2. Node.js npx 패키지 구동
Node.js 환경이 세팅되어 있는 경우 아래 명령으로 간편 구동이 가능합니다:
```bash
npx http-server ./ -p 8000
```
구동 후 브라우저에서 동일하게 `localhost:8000`으로 이동합니다.
